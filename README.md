# Driverforge Scoop bucket

Scoop bucket for [Driverforge](https://driverforge.dev) command-line tools:

- **driverforge** — build, deploy, and manage Control4 drivers
- **gayle** — deploy configuration and secrets to AWS SSM Parameter Store and
  Azure Key Vault

## Install

```
scoop bucket add driverforge https://github.com/driverforge/scoop-bucket
scoop install driverforge
scoop install gayle
```

## What's in this repository

This repository is intentionally minimal, so it's easy to review:

- It contains only the manifests for the tools listed above, generated by
  [GoReleaser](https://goreleaser.com) and committed automatically by each
  tool's release pipeline. No human pushes to this repository.
- The manifests download binaries from `releases.driverforge.com` (our
  release CDN), verified by the SHA-256 checksums pinned in each manifest —
  nothing is built or fetched from GitHub at install time.

## Upgrading

```
scoop update driverforge
scoop update gayle
```
