# Driverforge Scoop bucket

Scoop bucket for the [Driverforge](https://driverforge.dev) CLI — build,
deploy, and manage Control4 drivers.

## Install

```
scoop bucket add driverforge https://github.com/driverforge/scoop-bucket
scoop install driverforge
```

## What's in this repository

This repository is intentionally minimal, so it's easy to review:

- It contains only the `driverforge` manifest, generated by
  [GoReleaser](https://goreleaser.com) and committed automatically by our
  release pipeline on each release. No human pushes to this repository.
- The manifest downloads binaries from `releases.driverforge.com` (our
  release CDN), verified by the SHA-256 checksums pinned in the manifest —
  nothing is built or fetched from GitHub at install time.

## Upgrading

```
scoop update driverforge
```
